# Canon — 3-minute demo storyboard

Target: under 3:00, tightly edited, screen-recorded. Pre-generate both episodes beforehand so
nothing renders live on camera (the agent genuinely made them; you are not waiting on ffmpeg during
the take). Record the app at desktop width so the two-column layout shows.

| Time | On screen | Voiceover (the point being made) |
|---|---|---|
| 0:00–0:12 | The market: one line on the $16B AI micro-drama boom, then a jump-cut of typical AI clips where a character's look changes shot to shot. | "AI can generate a clip. It can't keep the same character across a series. That's the unsolved problem." |
| 0:12–0:20 | The Canon Studio, clean and still. Type a one-line premise, pick the style. | "Canon runs the whole showrunner job from one premise." |
| 0:20–1:00 | Click Generate. The Agents panel lights up step by step: Writer, Cinematographer, Generator, QC Critic, Editor. | "Four agents share one Story Bible. The Writer casts it, the Bible locks each character's look and seed." |
| 1:00–1:35 | Episode 1 plays in the screening room. | "Here's episode one. Coherent, voiced, subtitled." |
| 1:35–2:05 | The mic-drop: hit Generate again for episode 2. When it plays, cut side by side with episode 1 on the same character. | "Now episode two. Same character, same face, because it reused the Bible, not a new cast. This is the thing almost nobody else shows." |
| 2:05–2:30 | Zoom the QC Critic step: show it flagging a drifted shot and regenerating it. | "And it checks itself. Qwen-VL catches a shot that drifted and re-rolls it, capped." |
| 2:30–2:50 | The Story Bible cards: locked descriptors and seeds. Then a one-frame architecture diagram. | "Persistent canon plus a self-correcting critic. Built on Qwen, Qwen-VL, and Wan." |
| 2:50–3:00 | The deployed URL on Alibaba Cloud; the repo; "MIT, AI Showrunner track." | "Live on Alibaba Cloud, open source. That's Canon." |

## Do / don't
- **Do** lead with the problem and end on the episode-2 reveal; that is the whole pitch.
- **Do** keep the aesthetic stylized so nothing looks uncanny.
- **Don't** generate live on stage. Pre-render, then play.
- **Don't** narrate the plumbing. Judges reward the consistency story and the self-correction, not the file layout.
